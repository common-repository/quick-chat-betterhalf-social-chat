== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `quick-chat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The quick chat will be available on WordPress administration page.
